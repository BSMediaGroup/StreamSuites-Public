SYSTEM REQUIREMENTS

Built for modern Windows

  • Windows 10 build 19041 or later
  • 64-bit Windows installation
  • Compatible display and Windows Graphics Capture support
  • No Visual Studio or .NET SDK required by the self-contained installer

Capture and GPU behavior varies by device, display topology, driver, and Windows configuration. No unsupported minimum RAM or GPU claim is made.

CURRENT ALPHA

Native foundations, truthful boundaries.

StudioApp remains a native WPF/C++ application. Runtime/Auth owns accounts, rooms, permissions, credentials, and destinations; the supervised native engine owns local media work.

Available now

Account and room access, native display capture, Stage preview, detachable docks, destination configuration, updater, and local media foundations.

Not yet available

Live output, recording, clipping, and native remote-participant rendering are not implemented. ALPHA is not presented as production-stable.
