# StreamSuites-Public v0.5.0-alpha

`StreamSuites-Public v0.5.0-alpha` delivers major economy and live status foundations to the public surface while continuing FindMeHere directory polish, auth surface consistency, and release-facing stability work.

## 🛒 Economy & Market

- Full public Market & Exchange experience now live with catalog browsing, purchases, exchanges, and livechat command integration (`!market`, `!buy`, `!exchange`).
- Scoped wallet and inventory visibility with global roll-up behavior and creator-controlled suppression settings.
- Rich public item metadata (tooltips, descriptions, icons) powered by runtime authoritative definitions.

## 📡 Live Status & Kick

- Kick added as first-class live provider with official webhook support and real-time status on public profiles.
- Improved latest stream tray and ended stream history presentation using safe runtime data.
- `findmehere.live` surfaced as its own distinct analytics and alerting directory surface.

## 🔐 Auth & Identity

- Public identity assignment and secondary identity self-service unassign flows now available and consistent.
- Auth gate and session handoff improvements aligned with the broader platform.

## 🎨 Surface Polish

- FindMeHere directory experience, theming (light/dark), custom links, and branding refinements.
- Consistent hovercard, badge, and social platform registry behavior.

## ✅ Release & Packaging

- BUMP_NOTES.md prepared with 0.4.2-alpha archiving and new 0.5.0-alpha section.
- This changelog created as the authoritative release note source for the Public surface.
