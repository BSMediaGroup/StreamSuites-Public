Privacy policy (provisional)

Static surface

StreamSuites limits data collection on public pages. Creator authentication and secrets stay inside the dashboard runtime, never on these static views.

Data collection

We do not run trackers on public pages. Basic server logs and CDN metrics may capture IP, user agent, and timestamp solely for uptime and abuse detection.

  • Public HTML is static and cache-friendly.
  • No third-party analytics pixels are embedded.
  • Dashboard authentication happens in the protected admin area.

Content ownership

Creators retain rights to uploaded clips and metadata. Public previews reference media locations without exposing private storage keys.

  • Links point to platform-native destinations (YouTube, Twitch, Rumble).
  • Temporary thumbnails and artwork respect creator ownership.
  • Removal requests: privacy@stream.suites

Retention & controls

Logs that include personal data are purged on a rolling basis. Public caches may refresh slowly; critical removals are prioritized via manual invalidation.

  • Runtime keys and tokens are never shipped to the browser.
  • Dashboard exports stay within creator control.
  • Future updates will enumerate retention windows per region.
